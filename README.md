# parkinglotProject
주차장 정보 게시판 프로젝트

## 프로젝트 기획의도
운전자에게 있어 가장 큰 고충 중 하나는 주차공간 확보입니다. 조금이라도 붐비는 지역에는 늘상 '주차지옥'이 펼쳐지기 마련입니다. 
흔히들 주차공간이 부족하다고 말합니다. 
하지만 전국 주차장 확보율은 2017년 기준 98.53%이며 가장 주차난이 심각하다 여겨지는 서울시는 130%에 달하며, 차량 1대당 1.3대의 주차장소가 확보되어 있습니다. 
그러나 주차장소에 대한 충분한 정보가 제공되지 않아, 많은 운전자들이 주차공간 확보에 어려움을 겪곤 합니다. 
주차공간에 대한 충분한 정보를 제공받는다면, 원하는 자리에 어려움 없이 주차할 수 있을 것입니다.

본 프로젝트에서는 전국 주차장정보 공공데이터를 지도상에 구현하였습니다.
지역별 정보검색을 통해 운전자들이 한눈에 주변 주차장의 위치, 크기, 요금, 운영시간 등을 제공하는 게시판입니다. 
또한 웹에 가입한 회원들이 사용한 주차장에 대한 별점과 리뷰를 남김으로써, 자신에게 맞는 주차공간을 쾌적하게 이용할 수 있도록 도움을 제공하는 게시판을 기획하였습니다.


## 사용 기술
JAVA, SPRING, JAVA Script, AJAX, HTML, CSS, ORACLE DB 등등..  

## parkinglot 게시판 기능
- 목적지에 주변의 주차장 위치와 정보를 제공
- 전국 주차장 리스트를 확인하여 원하는 상세정보 검색 가능
- 카테고리를 선정하여 검색키워드를 통한 검색이 가능

## parkinglot review기능
- parkinglot detail 페이지에서 로그인 후 리뷰 및 별점(1점 ~ 5점) 입력 가능
- 로그인 상태에서 리뷰 삭제 가능

## 게시판 회원가입 및 로그인 기능 구현
- 회원가입 시 아이디, 패스워드, 닉네임, 전화번호를 입력하여 MEMBER 테이블에 insert하여 데이터를 관리
- 로그인/로그아웃을 구현하여 MYPAGE를 통해 추후 정보 수정이 가능

## 카카오 맵 API 이용
- 주차장 공공데이터의 정보를 JSON 형식으로 받아 카카오 맵에 marker를 띄워 상세정보 확인 가능
- 장소 검색을 통해 주변 주차장 정보 확인 가능
